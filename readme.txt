== Description ==
Templazee is a free WordPress plugin offering a collection of Gutenberg Blocks and a robust template library, designed for Full Site Editing. It simplifies website creation by allowing you to effortlessly add and customize blocks and templates in the new WordPress block editor. With its extensive range of page-building options, Templazee helps you build your ideal website quickly and efficiently.

== Installation ==  
= Using The WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Search for "Templazee"
* Click Install Now
* Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select templazee.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

= Using FTP =
* Download templazee.zip
* Extract the templazee directory to your computer
* Upload the templazee directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
